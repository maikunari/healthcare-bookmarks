Magic link bookmarking system for healthcare providers with email capture.

== Description ==

A WordPress plugin that provides magic link bookmarking functionality for healthcare providers with email capture for marketing.

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/healthcare-bookmarks/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings â†’ Healthcare Bookmarks screen to configure

== Frequently Asked Questions ==

= What post types does this work with? =
Currently only healthcare_provider post type.

= How long do magic links last? =
Magic links expire after 15 minutes for security.
